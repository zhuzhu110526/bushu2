## 前言
因为我把电脑装了win11（win10的git经常不能用）所以博客也不行了，这里我教大家怎么办
## 方法
### git安装
没什么好说的
### node.js安装
没什么好说的
### 安装hexo 
```
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
```
### 初始化git
获取密钥
在github更新
### 仓库配置
在配置文件中替换地址
### 部署到github
```
hexo g
hexo clean
hexo d
```
如果不行就
```
hexo d g
```
### 替换
把之前的文件替换掉现在的位置
```
hexo g
hexo d
hexo clean
```
